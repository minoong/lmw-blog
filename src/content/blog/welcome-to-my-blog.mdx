# Tech Blog에 오신 것을 환영합니다

안녕하세요! 새로운 기술 블로그를 시작하게 되어 기쁩니다.

## 블로그 소개

이 블로그는 개발하면서 배운 것들, 문제를 해결한 과정, 그리고 새로운 기술에 대한 탐구를 기록하는 공간입니다.

### 다룰 주제들

- **프론트엔드 개발**: React, Next.js, TypeScript 등
- **백엔드 개발**: Node.js, API 설계, 데이터베이스
- **DevOps**: CI/CD, 배포, 인프라
- **문제 해결**: 실제 프로젝트에서 마주친 이슈와 해결 과정

## 기술 스택

이 블로그는 다음 기술들로 만들어졌습니다:

```typescript
const techStack = {
  framework: 'Next.js 16',

```

