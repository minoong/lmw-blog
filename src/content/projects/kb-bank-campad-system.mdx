## 프로젝트 개요

비대면 카드 발급/신청 시 전송되는 첨부파일 및 처리건들을 전담직원을 지정하여 처리하고, 진행 상태를 확인할 수 있는 시스템을 구축했습니다. 영업직원 관리, 전담직원별 Todo-List 기반 업무 스케줄링, 첨부파일 관리 기능을 통해 비대면 카드 발급 프로세스의 효율성을 향상시켰습니다.

## 프로젝트 정보

- **고객사**: KB국민은행
- **포지션**: Front/Back-end
- **개발 인원**: 4명

## 기술 스택

- **Frontend**: 인젠트 단발 개발 툴
- **Backend**: Java, Spring Framework
- **Database**: Oracle

## 담당 업무

### 1. **CamPad 영업직원 관리**

- **영업직원 정보 관리**
  - 영업직원 등록/수정/삭제 기능 구현
  - 소속 지점, 팀, 직급 등 조직 정보 관리
  - 영업직원별 권한 설정 및 접근 제어
  - 명함 이미지 등록 및 관리

- **영업직원 검색 및 필터링**
  - 영업직원 검색 기능
  - 소속, 직급, 활동 상태별 필터링

### 2. **CamPad 전담직원별 배정 상태 구현**

- **배정 상태 관리**
  - 대기, 진행 중, 완료, 보류 등 상태별 분류
  - 상태 변경 이력 추적 및 로그 관리
  - 업무 인수인계 기능 (전담직원 변경 시)

### 3. **CamPad 첨부파일 관리**

- **첨부파일 업로드 및 저장**
  - 비대면 카드 신청 시 필요 서류 첨부 기능
  - 다중 파일 업로드 지원
  - 파일 형식 및 용량 검증
  - 파일 암호화 저장 및 보안 처리

- **첨부파일 조회 및 다운로드**
  - 신청 건별 첨부파일 목록 조회
  - 파일 미리보기 기능 (이미지, PDF 등)
  - 개별 파일 또는 일괄 다운로드
  - 다운로드 이력 관리

- **첨부파일 검증 및 승인**
  - 전담직원의 첨부파일 검토 및 승인 프로세스
  - 불완전한 서류에 대한 재제출 요청 기능
  - 첨부파일 승인/반려 이력 관리

- **첨부파일 보안 및 권한 관리**
  - 전담직원만 첨부파일 접근 가능하도록 권한 제어
  - 개인정보 포함 파일 암호화 처리

## 개발 구현 내용

### Frontend 개발

1. **인젠트 단발 개발 툴 기반 UI**
   - 인젠트 프레임워크를 활용한 화면 개발
   - 영업직원 관리 화면 구현
   - 배정 상태 UI 개발

2. **업무 대시보드**
   - 전담직원별 현재 처리 중인 업무 현황
   - 진행시간 및 예상 완료 시간 표시
   - 실적 현황 및 처리율 시각화

### Backend 개발

3. **Spring Framework**
   - Spring MVC 패턴
   - RESTful API 설계 및 구현

4. **영업직원 관리 시스템**
   - 영업직원 정보 CRUD 기능
   - 명함 이미지 업로드/관리 로직

5. **배정 상태 시스템**
   - 진행시간/처리시간 계산 엔진
   - 배정 상태 변경 및 이력 관리

6. **첨부파일 관리 시스템**
   - 멀티파트 파일 업로드 처리
   - 파일 암호화/복호화 로직
   - 파일 메타데이터 관리

### Database 개발

7. **Oracle DB**
   - 영업직원 정보 테이블 설계
   - 첨부파일 메타데이터 테이블 설계
   - 상태 변경 이력 및 로그 테이블 설계

### 시스템 최적화

8. **성능 최적화**
   - 쿼리 성능 튜닝 및 인덱스 설계
