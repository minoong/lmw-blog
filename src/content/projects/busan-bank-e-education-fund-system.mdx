## 프로젝트 개요

e-교육금고는 공무원의 지출·재무 업무를 은행과 연계해 처리하는 전자시스템으로, 공무원이 은행 방문 없이 내부망을 통해 직접 은행 업무를 볼 수 있게 하는 시스템입니다. e-교육금고 시스템으로 교육청의 재정 업무를 개선, 지출 관련 전자결재 시스템을 구축했습니다.

## 프로젝트 정보

- **고객사**: 부산은행, 부산시교육청
- **포지션**: Front/Back-end
- **개발 인원**: 5명

## 기술 스택

- **Frontend**: JavaScript, jQuery, jqGrid, High Chart
- **Backend**: Java, Spring Framework
- **Database**: Oracle
- **Server**: Linux, Windows

## 담당 업무

### 1. **jqGrid 공통 모듈화**

jqGrid를 프로젝트 전반에서 효율적으로 사용하기 위해 공통 모듈로 확장하고, 다양한 기능을 추가했습니다.

#### **FormData를 이용한 AJAX 기능 확장**
- 기존 jqGrid의 AJAX 통신 방식을 FormData로 확장
- 파일 업로드 및 복합 데이터 전송 지원
- multipart/form-data 방식의 데이터 전송 처리
- AJAX 요청/응답 인터셉터 구현

#### **무한 스크롤 방식으로 기능 확장**
- 기존 pager 단위 페이지 처리 방식에서 무한 스크롤 방식으로 전환
- 스크롤 이벤트 감지 및 자동 데이터 로딩
- 로딩 인디케이터 표시 및 UX 개선

#### **LocalStorage를 활용한 상태값 저장**
- 페이지 뒤로 이동 시 그리드 상태 복원
- 정렬, 필터, 검색 조건 저장

### 2. **사용자 UI 컴포넌트 구현 (jQuery 플러그인 확장)**

SPA(Single Page Application)와 같은 사용자 경험을 제공하기 위해 화면을 Top, Left, Main 3가지 영역으로 나누고, 모든 상태값을 연동하는 커스텀 UI 프레임워크를 구축했습니다.

#### **탭 기반 메뉴 시스템**
- Left 메뉴의 하위 각 메뉴를 Main 화면의 탭으로 구성
- 다중 탭 동시 오픈 및 전환 기능
- 탭 닫기, 전체 닫기, 현재 탭 외 모두 닫기 기능
- 탭 드래그 앤 드롭으로 순서 변경

#### **3가지 영역 상태 동기화**
- Top, Left, Main 영역의 상태값을 실시간 동기화
- 메뉴 선택 시 Left 메뉴 활성화 상태와 Main 탭 동기화
- Breadcrumb 자동 생성 및 업데이트
- 이벤트 기반 상태 전파 메커니즘 구현

#### **자주 사용되는 메뉴 자동 즐겨찾기**
- 사용자의 메뉴 사용 빈도 자동 추적
- 사용 횟수 기반 자동 즐겨찾기 등록
- 즐겨찾기 메뉴 빠른 접근 UI 제공하기

### 3. **Admin Dashboard 구현 (jQuery 플러그인 확장)**

관리자를 위한 대시보드를 jQuery 플러그인 형태로 확장하여 재사용 가능한 컴포넌트로 구현했습니다.

- **위젯 시스템**
  - 드래그 앤 드롭으로 위젯 배치 변경

- **실시간 모니터링**
  - 업무 처리 현황 통계

- **알림 시스템**
  - 중요 이벤트 알림 표시
  - 미처리 업무 알림
  - 시스템 공지사항

### 4. **High Chart를 이용한 시각화 처리**

교육청의 재정 업무 데이터를 효과적으로 시각화하여 의사결정을 지원했습니다.

- **원장 데이터 시각화**
  - 예산 집행 현황 차트 (막대, 선형, 원형)
  - 월별/분기별 지출 추이 분석
