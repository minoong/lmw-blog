## 프로젝트 개요

법인지방소득세의 지방세를 납세자가 스스로 세액을 계산하여 신고 납부서 작성 및 납부 시스템을 통하여 신고납부할 수 있는 시스템을 구축했습니다.

## 프로젝트 정보

- **개발 기간**: 2014.12 ~ 2015.05
- **고객사**: 부산은행
- **포지션**: Front/Back-end
- **개발 인원**: 6명

## 기술 스택

- **Frontend**: X-Platform
- **Backend**: Java, Spring Framework
- **Database**: Oracle
- **Server**: Linux, Windows

## 담당 업무

1. **신고내역 조회**
   - 납세자의 과거 신고내역을 조회할 수 있는 기능 개발

2. **세액 미리 계산하기**
   - 납세자가 신고 전에 예상 세액을 미리 계산할 수 있는 기능 개발

3. **납세자 관리대장 서비스**
   - 납세자 정보를 체계적으로 관리할 수 있는 관리대장 페이지 개발

## 개발 구현 내용

### Frontend 개발

1. **X-Platform 기반 UI 개발**
   - X-Platform을 활용한 크로스 플랫폼 클라이언트 애플리케이션 개발
   - Dataset을 활용한 데이터 바인딩 및 상태 관리

2. **화면 설계 및 구현**
   - 납세 신고 입력 폼 및 유효성 검증 로직 구현
   - 신고내역 조회 및 검색 기능 개발

### Backend 개발

3. **Spring Framework 기반 서비스 레이어**
   - Spring MVC 패턴을 활용한 계층형 아키텍처 구축
   - RESTful 방식의 Controller 설계

4. **비즈니스 로직 구현**
   - 지방세 세액 계산 로직 개발
   - 전자납부 전문 송수신 처리 로직
   - 납세자 관리대장 CRUD 서비스 구현

### Database 개발

5. **Oracle DB**
   - 납세자, 신고내역, 납부내역 등 테이블 조회 쿼리 작성

### 시스템 연동

6. **은행 시스템 인터페이스**
   - 부산은행 전문 통신 프로토콜 연동
   - 전문 파싱 및 전문 생성
   - 전문 송수신 이력 관리 및 에러 핸들링
