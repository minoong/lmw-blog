# IoCare+ 서비스 런칭

## 프로젝트 개요

코웨이의 새로운 IoT 기반 헬스케어 서비스인 IoCare+를 개발하고 런칭했습니다.

## 주요 기능

- 실시간 건강 데이터 모니터링
- AI 기반 건강 분석 및 추천
- 가족 건강 관리 기능

## 기술 스택

- **Frontend**: Next.js, React, TypeScript
- **Backend**: Node.js, Express
- **Database**: PostgreSQL
- **Deployment**: Vercel, GitHub Actions

## 성과

- 첫 달 사용자 수 10,000명 달성
- 평균 사용자 만족도 4.8/5.0
- 월간 활성 사용자 증가율 150%
