## 프로젝트 소개

Next.js 16과 MDX를 활용하여 만든 개인 기술 블로그입니다. Nextra의 블로그 테마를 참고하여 구현했으며, 마크다운으로 간편하게 콘텐츠를 작성할 수 있습니다.

## 주요 기능

- MDX 기반 콘텐츠 작성
- 코드 하이라이팅
- 태그 및 카테고리 시스템
- 반응형 디자인
- SEO 최적화

## 기술 스택

- Next.js 16 (App Router)
- TypeScript
- MDX
- Tailwind CSS
- Gray Matter

## 배운 점

- Next.js App Router의 파일 기반 라우팅 구조
- MDX를 활용한 동적 콘텐츠 관리
- 정적 사이트 생성(SSG)을 통한 성능 최적화
